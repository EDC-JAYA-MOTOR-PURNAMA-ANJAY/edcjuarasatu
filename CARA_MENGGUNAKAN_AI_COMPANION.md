## ðŸ”§ **TROUBLESHOOTING**

### **Problem: AI tidak merespon**

**Solution:**
1. Check internet connection
2. Refresh page (F5)
3. Check console (F12 â†’ Console tab)
4. Screenshot error â†’ Report ke admin

### **Problem: Response lambat (>10 detik)**

**Solution:**
1. Internet mungkin lambat
2. API quota mungkin limit (tunggu 1 menit)
3. Try again

### **Problem: Error message muncul**

**Solution:**
1. Screenshot error
2. Copy error message
3. Report ke admin/developer
4. Sementara coba akses ulang

### **Problem: Halaman tidak bisa diakses (404)**

**Solution:**
1. Pastikan URL benar: `/student/ai-companion`
2. Pastikan sudah login sebagai SISWA
3. Clear browser cache (Ctrl + Shift + Del)
4. Try again
